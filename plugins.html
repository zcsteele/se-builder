Plugins

Want to extend Selenium Builder functionlity? Here is how you do it!

You can manage your plugins directly from within Selenium Builder. If you want to develop a plugin, have a look at the developer documentation.

Selenium Builder also supports adding basic new steps via a user_extensions.js file. To install the file, put it in 

<your firefox profile folder>/SeBuilder/user-extensions.js
(Note that this only supports extensions that add steps, not ones that add new user interface such as flow-control.)
